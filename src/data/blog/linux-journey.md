# Linux Journey




Learning Linux is something a developer or every security enthusiast would (or should) love to do and be comfortable with, in their career. The most common reasons for this include the wide spread use of the platform and ease of tools available for various use cases.

I recently visited a website [Linux Journey](https://linuxjourney.com/) recommended by a friend, that helps in learning Linux. Since then, I've been trying to follow along and refresh my  *Linux-fu* as they call it. The journey is broken down into 3 main sections:

- Grasshopper
- Journeyman
- Networking nomad

These sections helps you start from basic Linux skills such as command line, text editing, user management and system processes to networking topics such as DNS, routing and subnetting on Linux. There are good examples to try out on our systems for hands own understanding and also features an exercise and a quiz section to test your understanding of a particular topic. The site also features references to other Linux learning resources.

Hope you find this site a good refresher to hone your Linux skills.
